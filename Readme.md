# Conversational Search with Knowledge Graph (FalkorDB)

This project implements a **Knowledge Graph service** using **FastAPI** and **FalkorDB**.
It supports conversational search features like constraint resolution, recommendations,
and explainability using Cypher queries.

## Tech Stack
- FastAPI
- FalkorDB (Cypher-compatible graph database)
- Docker & Docker Compose

## Run Instructions

```bash
docker compose up --build
